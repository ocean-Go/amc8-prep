# AMC8 备考系统 - MVP

## 项目设置

### 1. 初始化 Next.js 项目
```bash
npx create-next-app@latest amc8-prep --typescript --tailwind --app
cd amc8-prep
npm install @supabase/supabase-js @supabase/ssr
```

### 2. 环境变量 (.env.local)
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
MINIMAX_API_KEY=your_minimax_api_key
```

### 3. Supabase 表结构
```sql
-- users 表
create table users (
  id uuid default uuid_generate_v4() primary key,
  name text not null,
  age integer not null,
  avatar text,
  created_at timestamp with time zone default timezone('utc'::text, now())
);

-- progress 表
create table progress (
  id bigint generated by default as identity primary key,
  user_id uuid references users(id),
  topic text not null,
  completed boolean default false,
  score integer default 0,
  updated_at timestamp with time zone default timezone('utc'::text, now())
);

-- wrong_answers 表
create table wrong_answers (
  id bigint generated by default as identity primary key,
  user_id uuid references users(id),
  question_id text not null,
  question_text text not null,
  user_answer text,
  correct_answer text,
  error_type text,
  times_wrong integer default 1,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

### 5. 核心功能

#### 首页 (app/page.tsx)
- 登录/注册按钮
- 两个孩子的入口 (Matt / Chris)
- 简洁友好的界面

#### 学习页面 (app/topics/[topic]/page.tsx)
- 知识点讲解
- 示例题目
- 练习按钮

#### 练习页面 (app/practice/page.tsx)
- 随机题目
- 计时器
- 即时反馈

#### 错题本
- 自动收集错题
- 按知识点分类
- AI 讲解 (MiniMax)

### 6. AI 讲解 API (app/api/ai-explain/route.ts)
```typescript
import { MiniMax } from '@/lib/minimax';

export async function POST(request: Request) {
  const { question, wrongAnswer, userId } = await request.json();
  
  const prompt = `解释这道AMC8题目错在哪里：\n题目：${question}\n错误答案：${wrongAnswer}`;
  
  const response = await MiniMax.chat.completions.create({
    model: 'MiniMax-M2.5',
    messages: [{ role: 'user', content: prompt }]
  });
  
  return Response.json({ explanation: response.choices[0].message.content });
}
```
